Code to be executed during the 15 second autonomous period. 

\par
The goal is to knock both of the large balls off of the rack into the scoring zone. \par
Instructions\-:\par

\begin{DoxyEnumerate}
\item Raise the lift above the 12 in. bar. 
\item Orient the robot so that it is facing the large ball nearest to it. 
\item When the autonomous period starts, the robot will go forward and knock the large ball into the scoring zone. Then, it will return to its starting position. Once the robot gets back to the starting position, you have five seconds to rotate it so that it is facing the second large ball. 
\end{DoxyEnumerate}After that the robot will go forward and knock the second large ball off of the rack. It will remain there until user control starts. \hypertarget{_competition1_8c_a2682bd0e6b87fc110fb4a0d6632ffa02}{\index{Competition1.\-c@{Competition1.\-c}!move\-Lift@{move\-Lift}}
